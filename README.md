# manual-testing-login-module
This repository contains a manual testing project for the Swag Labs login page. It includes well-structured test cases covering positive, negative, and UI scenarios documented in an Excel sheet. Screenshots are attached as evidence, and bugs such as non-functional UI elements are identified to ensure system reliability and usability.

# Objective: Validate functionality, usability, and reliability of the login system
1. Test cases designed for positive, negative, boundary, and UI scenarios
2. All test cases documented in a structured Excel sheet
3. Each test case includes steps, expected results, actual results, and pass/fail status
4. Screenshots attached as evidence of test execution
5. UI and functional bugs identified and documented (e.g., non-working elements)
6. Focus on real-world SQA practices and testing techniques
7. Demonstrates skills in test case design, bug reporting, and result analysis
